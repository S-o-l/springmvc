Request mapping in Spring 3 MVC

Mapping by path:
Spring MVC dispatcher request mapping by path with @RequestMapping("path") annotation

Mapping by path

This is example of mapping by path in Spring 3 MVC


Code of mapping by path

This is code of mapping by path usage in Spring 3 MVC

@RequestMapping("mapbypath")
public @ResponseBody String getByPath() {
return "Hello, this is request to /mvc/requestmap/bypath path";
}
Mapping by method:
Spring MVC dispatcher request mapping by method (usually GET/POST) with @RequestMapping(method=value) annotation

Mapping by method

This is example of mapping by method in Spring 3 MVC


Make a request with a method to "mvc/requestmap/bymeth":

Code of mapping by method

This is code of mapping by method usage in Spring 3 MVC

@RequestMapping(value="mapbymeth", method=RequestMethod.GET)
public @ResponseBody String getByMethGet() {
return "Hello, this is GET request to /mvc/requestmap/mapbymeth path";
}

@RequestMapping(value="mapbymeth", method=RequestMethod.POST)
public @ResponseBody String getByMethPost() {
return "Hello, this is POST request to /mvc/requestmap/mapbymeth path";
}
Mapping by presence of parameter:
Spring MVC dispatcher request mapping by value with @RequestMapping(params="value") annotation

Mapping by presence of parameter

This is example of mapping by presence of parameter in Spring 3 MVC


Make a request with a 'goodparam' or 'badparam' to check method to "mvc/requestmap/mapbypresparam":

Code of mapping by presence of parameter

This is code of mapping by presence of parameter usage in Spring 3 MVC

@RequestMapping(value="mapbypresparam", params="goodparam")
public @ResponseBody String getByPresParam() {
return "Hello, this is request with presence of 'goodparam' parameter";
}
Mapping by absence of parameter:
Spring MVC dispatcher request mapping by absence of value with @RequestMapping(params="!value") annotation

Mapping by absence of parameter

This is example of mapping by absence of parameter in Spring 3 MVC

Code of mapping by absence of parameter

This is code of mapping by absence of parameter usage in Spring 3 MVC

@RequestMapping(value="mapbyabsparam", params="!foo")

return "Hello, this is request with absence of 'foo' parameter";
}

Obtaining request data in Spring 3 MVC

Request parameter:
Obtaining request parameter in Spring MVC with @RequestParam annotation

Obtaining request parameter

This is example of obtaining of request parameter in Spring 3 MVC



Code of obtaining of request parameter

This is code of request parameter in Spring 3 MVC

@RequestMapping("reqparam")
public @ResponseBody String reqParam(@RequestParam String parameter) {
return "Hello, this is a request with a parameter and its value is " + parameter;
}
Group of parameters:
Obtaining group of parameters from request in Spring MVC with '(Object object)' method parameters

Obtaining group of parameters

This is example of obtaining of group of parameters in Spring 3 MVC





Code of obtaining group of parameters

This is code of obtaining group of parameters in Spring 3 MVC

@RequestMapping("groupparam")
public @ResponseBody String reqParam(GroupParam groupparam) {
return "Hello, this is a request with a group of parameters
and they are: " + groupparam.getParam1() +
" and " + groupparam.getParam2() +
" and " + groupparam.getParam3();
}

Obtaining part of path as variable
